### API功能测试脚本
```bash
#!/bin/bash
echo "🧪 开始API功能测试..."

# 测试提交灵感
echo "🔍 测试提交灵感..."
response=$(curl -s -X POST http://localhost:3001/api/v1/ideas \
  -H "Content-Type: application/json" \
  -d '{"openid":"test_openid","textContent":"测试灵感"}')

    echo "✅ 提交灵感成功"

else
    echo "❌ 获取市场列表失败: $market_response"
    exit 1
fi

echo "🎉 API功能测试通过！"
```

## 🔧 故障排除指南

### 常见问题及解决方案

**问题1: 服务无法启动**
- 检查端口占用: `lsof -i :3001`
- 重启服务: `pkill -f node && npm start`

**问题2: 微信JS-SDK初始化失败**  
- 确认域名已添加到JS接口安全域名
- 检查AppID和AppSecret配置

**问题3: API返回404错误**
- 确认后端服务正在运行
- 检查API路由配置

**问题4: 图片/语音上传失败**
- 检查文件大小限制（10MB）
- 确认浏览器支持相关API
